
# AI Features - Quick Reference

## 🚀 Quick Start

### First Time Setup (2 minutes)
1. Open app → Profile → AI Configuration
2. Choose autonomy level (start with "Suggest-Only")
3. Explore with demo mode (no API keys needed)
4. Add API keys later for real functionality

---

## 🧠 Problem Solver

### Quick Actions
- **Analyze Problem**: Paste text → Tap "Analyze Problem"
- **View Solutions**: Switch to Solutions tab
- **Rate Solution**: Tap thumbs up/down
- **Execute**: Tap "Execute Solution"

### Keyboard Shortcuts
- `Cmd/Ctrl + V`: Paste problem
- `Enter`: Analyze
- `↑/↓`: Navigate solutions
- `Space`: Rate current solution

### Tips
- Include deadlines in problem description
- Rate every solution to improve AI
- Connect productivity tools for auto-logging
- Check impact scores (80+ = high impact)

---

## ✈️ Travel Assistant

### Quick Search
```
Type: "Meeting in Berlin tomorrow"
Tap: "Find Options"
Review: Ranked results
Select: Preferred option
Confirm: Book
```

### Tabs Overview
- **Trips**: Detected travel needs
- **Options**: Available flights/hotels
- **Plan**: Complete itinerary
- **Live**: Real-time tracking

### Ranking Indicators
- 🟢 Green leaf: High sustainability
- 🟡 Yellow leaf: Medium sustainability
- 🔴 Red leaf: Low sustainability
- ⭐ Stars: Provider rating
- 💰 Price: Total cost

### Tips
- Use natural language for search
- Check sustainability scores
- Review buffer times
- Enable trip tracking
- Monitor live dashboard during travel

## ⚙️ Configuration

### Autonomy Levels

| Level | Control | Auto-Actions | Best For |
|-------|---------|--------------|----------|
| Suggest-Only | Maximum | None | First-time users |
| Semi-Autonomous | Balanced | Routine tasks | Regular users |
| Fully Autonomous | Minimum | Everything | Power users |

### Quick Settings
- **Profile → AI Configuration → Autonomy**: Choose level
- **Profile → AI Configuration → APIs**: Add keys
- **Profile → AI Configuration → Preferences**: Set options

## 📊 Understanding Scores

### Impact Score (Problem Solver)
- **90-100**: Critical impact, do immediately
- **70-89**: High impact, prioritize
- **50-69**: Medium impact, schedule
- **30-49**: Low impact, consider
- **0-29**: Minimal impact, optional

### Effort Level
- **Low**: <1 hour, simple
- **Medium**: 1-4 hours, moderate
- **High**: >4 hours, complex

### Sustainability Score (Travel)
- **High**: <50kg CO₂, eco-friendly
- **Medium**: 50-150kg CO₂, average
- **Low**: >150kg CO₂, high carbon

## 🛠️ Troubleshooting

### Problem: AI Not Responding
**Fix:** Check API key → Verify internet → Try again

### Problem: No Travel Options
**Fix:** Check spelling → Verify dates → Try different provider

### Problem: Booking Failed
**Fix:** Check payment method → Verify API permissions → Contact support

### Problem: Calendar Not Updating
**Fix:** Grant calendar permission → Enable auto-update → Force sync
