
# Voice & Communication - Quick Start Guide

Get your AI assistant speaking and listening in 5 minutes!

---

## Step 1: Get Your API Keys (2 minutes)

### For Speech-to-Text (Choose One)

**OpenAI Whisper** (Recommended for beginners)
1. Go to https://platform.openai.com/api-keys
2. Click "Create new secret key"
3. Copy the key (starts with `sk-`)
4. Cost: ~$0.006/minute

**Deepgram** (Best for real-time)
1. Go to https://console.deepgram.com/
2. Sign up for free account
3. Get API key from dashboard
4. Free tier: 45,000 minutes/month

### For Text-to-Speech (Choose One)

**OpenAI TTS** (Recommended)
1. Same API key as Whisper above
2. Cost: ~$15 per 1M characters
3. 6 natural voices included

**ElevenLabs** (Best quality)
1. Go to https://elevenlabs.io/
2. Sign up for free account
3. Get API key from settings
4. Free tier: 10,000 characters/month

### For Calls (Optional)

**Twilio** (Most popular)
1. Go to https://www.twilio.com/try-twilio
2. Sign up for free trial ($15 credit)
3. Get Account SID and Auth Token
4. Buy a phone number ($1/month)

---

## Step 2: Configure API Keys (1 minute)

1. Open the app
2. Go to **Profile** → **AI Configuration**
3. Tap **APIs** tab
4. Find **Voice & Communication APIs** section
5. Toggle on your chosen provider
6. Paste your API key
7. Tap the checkmark to save

---

## Step 3: Enable Voice Features (1 minute)

1. Go to **Voice & Communication** tab (waveform icon)
2. Toggle on the features you want:
   - ✅ **Speech-to-Text**: Convert audio to text
   - ✅ **Text-to-Speech**: Generate voice replies
   - ⚠️ **Voice Cloning**: Create your custom voice (requires consent)
   - ⚠️ **Real-time Voice**: Live conversations
   - ⚠️ **Call Integration**: Handle phone calls

---

## Step 4: Test It Out (1 minute)

### Test Speech-to-Text
1. Scroll to **Speech-to-Text Test** section
2. Tap **Test Speech-to-Text**
3. Allow microphone access
4. Speak for 5 seconds
5. See your transcription!

### Test Text-to-Speech
1. Scroll to **Text-to-Speech Test** section
2. Edit the sample text or write your own
3. Tap **Generate Speech**
4. Listen to the result!

---

## Step 5: Configure Your Agent (Optional)

1. Go to **Agent** tab
2. Toggle on **Handle Calls**
3. Tap **Voice Settings**
4. Choose your preferred voice profile
5. Set response style (Professional, Friendly, etc.)

## Tips for Best Results

### For Speech-to-Text
✅ Speak clearly and at normal pace
✅ Use good microphone
✅ Minimize background noise
✅ Specify correct language

### For Text-to-Speech
✅ Use natural, conversational text
✅ Add punctuation for pauses
✅ Test different voices
✅ Adjust speed if needed

### For Voice Cloning
⚠️ Only clone your own voice
⚠️ Get explicit consent if using for others
⚠️ Record in quiet environment
⚠️ Use consistent microphone

## Troubleshooting

### "API Key Invalid"
- Check you copied the full key
- Verify key is active in provider dashboard
- Try regenerating the key

### "Microphone Not Working"
- Check app permissions in Settings
- Restart the app
- Test microphone in another app

### "Poor Transcription Quality"
- Speak more clearly
- Reduce background noise
- Try different STT provider
- Check internet connection

### "Voice Sounds Robotic"
- Try different TTS provider
- Adjust speed/pitch settings
- Use more natural text
- Consider ElevenLabs for best quality

Before enabling voice features, review:

✅ **Data Storage**: Where are recordings stored?
✅ **Retention**: How long are transcripts kept?
✅ **Sharing**: Is data shared with third parties?
✅ **Consent**: Have you given explicit consent?
